How tos

Features

  • Access your media entertainment anyplace in your home

Openning main webpage

Using desktop or Laptops

  1. Open your Internet Browser (Google Chrome or Firefox)

  2. In the address bar type in blox.local/

Using smartphones or tablets

  1. Open your Internet Browser (Chrome or stock browser)

  2. In the address bar type in 0.0.0.0

Connecting to your file server

There are 3 ways to access your file server

Using Browser

  1. Open the Blox's webpage and click on Owncloud

using file browser in windows 7 or 8

  1. On your desktop, press start

  2. Type \\blox.local

Using android application

  1. Download and install ES File Explorer from the google appstore

  2. Open the app and go to lan settings

  3. In the host field put \\192.168.0.254
